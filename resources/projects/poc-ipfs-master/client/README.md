# IOTA IPFS Client

This package provides a React Web UI to provide facilities to upload and authenticate files using IOTA and IPFS.

## Pre-requisites

```shell
npm install
```

## Development

This will run the Web UI at <http://localhost:3000>

```shell
npm run start
```
